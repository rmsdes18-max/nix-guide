# NIX — Throne and Liberty datamine guide

A single-page, black-and-white guide to everything datamined for the upcoming
**Nix** expansion in Throne and Liberty: new Archboss weapons, gear items,
enchantments and field items across all 10 weapons.

**Everything here is datamined and subject to change.**

## Source

Content extracted from a [Hexapod video](https://www.youtube.com/watch?v=FYPSsX68J8E),
in collaboration with K9 Beat & Gros. Unofficial fan page — Throne and Liberty
© NCSOFT / Amazon Games.

## Build

The page is generated from `content.md` by a small Node script:

```bash
node build.js   # writes index.html
```

Static files (`index.html`, `style.css`, `script.js`) need no build step to view —
just open `index.html` in a browser.
